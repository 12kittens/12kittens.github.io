Who am I?

I'm a hobby game developer and software programmer, actually 15 years old. My real name is Joaquín (I use my real name for my project's copyright).

I like to code apps and libraries under licenses like GPL, because I agree with the GNU Project (there are some things that should be free) and also because I don't want to make money from my hobby (yet). I develop web videogames, too.

I know HTML, CSS, JavaScript (both browser and Node.js), Python and some C++, PHP and React (but I'll always prefer Vanilla). I speak Spanish and I'm studying English and some Chinese. (My Chinese is even worse than my English.)

My projects

I actually have one published web videogame, Kittens Dash, and two in development, Fly Kitten Fly! and Kittens Temple. I'm also working on a free studying app whose name I haven't decided yet and a portable password manager (also GPL), called Nano PassBox, which I'm actually deploying.

I did also code two libraries: an unofficial Newgrounds.io bridge with modern web technologies (like SubtleCrypto and promises) and a lightweight videogame library, Sprite.js, which includes features for easily positioning 2D sprites in the canvas. (If you've ever tried to make a videogame using HTML5 canvases without external libraries, you would understand me. When I made that library I didn't know about PixiJS). I'm actually working on other things more related to Node.js, like a native multi-platform Node.js TTS engine.

Collaborating

I just code for fun. By now I'm not interested in collaborating with other developers. If you want to help with one of my projects (or you've found a bug and you fixed it by yourself), you can just leave the pull request and I'll try to upload it when possible, thanks. Depending on the license, you might also deploy your version by yourself.
